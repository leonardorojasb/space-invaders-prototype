## Descripción

Este repositorio contiene el prototipo inicial de un videojuego web inspirado en Space Invaders.

El proyecto fue desarrollado como una primera aproximación a las mecánicas principales del juego, incluyendo movimiento del jugador, disparos, enemigos, colisiones y sistema de puntuación.

Esta versión sirvió como base para desarrollar posteriormente una versión más completa y mejorada.

---

## Características

- Movimiento del jugador
- Sistema de disparos
- Movimiento de enemigos
- Detección de colisiones
- Sistema de puntuación
- Integración básica de imágenes y sonidos
- Ejecución directa en el navegador

## Ejecución local

### Visual Studio Code

1. Abrir el proyecto en Visual Studio Code.
2. Instalar la extensión **Live Server**.
3. Abrir `index.html` mediante:

```text
Open with Live Server
```

### Python

Desde la carpeta del proyecto, ejecutar:

```bash
python -m http.server
```

Después abrir:

```text
http://localhost:8000
```
